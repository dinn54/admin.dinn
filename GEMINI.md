## UI / Design System Rules

- Use shadcn/ui with the **base preset**
- Framework is **Next.js (App Router)** using the `next` template
- UI style preset is **mira**
- Base color palette is **neutral**
- Primary theme color is **emerald**
- Menu color is **default** with **subtle accent**
- Border radius uses shadcn/ui **default radius**
- Font must be **Noto Sans**
- Icon library must be **hugeicons**

All UI components and pages must strictly follow these settings.
Do not introduce other color themes, fonts, radius scales, or icon libraries.
